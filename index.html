대표 강의 & 프로젝트

PHYSICAL

A-1 피지컬 컴퓨팅 기초

대상 : 중등 / 고등 · 총 6차시

  • 1차시 : 아두이노 소개 및 LED 제어
  • 2차시 : 센서 입력과 출력 연결
  • 3차시 : 간단한 프로젝트 제작
  • 4차시 : 피지컬 컴퓨팅 응용 사례
  • 5차시 : 팀 프로젝트 설계
  • 6차시 : 발표 및 피드백
BLOCK CODING

B-1 블록코딩 창의 프로젝트

대상 : 초등 / 중등 · 총 4차시

  • 1차시 : 스크래치 기본 문법
  • 2차시 : 캐릭터 움직임과 이벤트 처리
  • 3차시 : 게임 / 스토리텔링 제작
  • 4차시 : 완성작 발표 및 공유
AI AUTOMATION
